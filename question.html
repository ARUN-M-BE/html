1. What are HTML5 Semantic Tags?

Answer

Semantic tags in HTML5 like <article>, <section>, <header>, <footer>, <nav>, and <aside> describe the meaning of the content, improving SEO and accessibility.

2. How to Embed Audio in HTML5?

Answer

Use the <audio> tag with controls and a <source> element inside.

3. How to Embed Video in HTML5?

Answer

Use the <video> tag with controls and one or more <source> elements for supported formats.

4. How to Use Canvas in HTML5?

Answer

Use the <canvas> tag and JavaScript to draw graphics using its 2D or WebGL context.

5. What is SVG?

Answer

<svg> is a tag used to define scalable vector graphics with elements like <circle>, <rect>, <path>.

6. What is the use of <details> and <summary>?

Answer

These tags create expandable sections that show or hide additional information when clicked.

7. What is the <progress> tag?

Answer

Displays progress of a task. Example: <progress value="50" max="100">.

8. What is the <meter> tag?

Answer

Displays a scalar measurement within a known range. Example: battery level, score, etc.

9. What are HTML5 Input Types?

Answer

New input types: email, date, datetime-local, month, number, range, tel, search, url, week, color.

10. What is the <datalist> tag used for?

18. What does the <mark> tag do?

19. What is the <time> tag?

Answer

Defines a specific time or date, useful for machine-readable dates.

20. What is the <figure> tag?

Answer

Used to encapsulate media content like images, diagrams, or illustrations with a caption.

21. What is the <nav> tag?

Answer

Defines navigation links (menus, sidebars, etc.) and helps screen readers and SEO identify navigation.

22. How to group multiple headings in HTML5?

Use the <hgroup> tag (deprecated in HTML5.1) to group heading levels.

23. What is the use of the <aside> tag?

Defines content indirectly related to the main content (like ads, quotes, or side notes).

🎨 CSS Q&A: Pseudo-classes, Combinators, Functions

15. What are CSS Combinators?

Answer

Used to define relationships between selectors: descendant ( ), child ( > ), adjacent ( + ), general ( ~ ).

16. What are CSS Functions?

Answer

Examples include: var(), calc(), rgb(), url(), attr().

13. What are Pseudo-classes in CSS?

Answer

They define the special state of an element. Example: :hover, :focus, :first-child.

14. What are Pseudo-elements in CSS?

Answer

They style parts of elements like ::before, ::after, ::first-letter.

1. What is a CSS pseudo-class?

Answer

A pseudo-class defines the special state of an element like :hover, :focus, :nth-child().

2. What is a CSS combinator?

Answer

CSS combinators describe relationships between selectors: (descendant), > (child), + (adjacent sibling), and ~ (general sibling).

3. What does :hover do?

Answer

It applies styles when a user hovers the mouse over an element.

4. How does :nth-child() work?

Answer

It targets elements based on their order inside a parent, e.g., :nth-child(2) targets the second child.

5. What is the difference between > and in CSS?

Answer

> targets direct children only. targets any descendant elements.

6. What is :root in CSS?

Answer

It selects the root element (usually <html>) and is used to define global variables.

7. What is var() function in CSS?

Answer

It is used to reference CSS custom properties (variables) defined using :root.

8. What does ::before do?

Answer

It inserts content before an element's actual content using content: "".

9. Difference between ::before and :before?

Answer

::before is the modern syntax, :before is old but still supported. Both work the same.

10. What does :not() do?

Answer

It selects all elements except the one(s) specified inside the parentheses.

11. What is an array in JavaScript?

Answer

An array is a list-like object that stores multiple values in a single variable.

11. What is AJAX?

Answer

AJAX (Asynchronous JavaScript and XML) lets you fetch and send data to a server without reloading the page.

12. How to perform AJAX using Fetch API?

Answer

Use fetch("url").then(res => res.json()).then(data => console.log(data)); for modern async AJAX.

12. How to create a JavaScript object?

Answer

Use const obj = { key: value } syntax to define objects.

13. How to loop through an array?

Answer

You can use for, forEach(), map() to iterate through an array.

14. How to define a function in JS?

Answer

Use function name() {} or arrow syntax const fn = () => {}.

15. What is a nested array?

Answer

An array inside another array. Example: [[1, 2], [3, 4]].

16. What is a nested object?

Answer

An object inside another object. Example: { user: { name: "Arun" } }.

17. What is the difference between map() and forEach()?

Answer

map() returns a new array, forEach() just loops through without return.

18. What is a method?

Answer

A method is a function defined inside an object: obj.greet = function() {}

19. How to fetch data using JavaScript?

Answer

Use fetch('url').then(...) to perform AJAX requests.

20. What is the use of JSON.parse() and JSON.stringify()?

Answer

JSON.parse() converts JSON strings into JS objects. JSON.stringify() does the reverse.

21. How to access object properties?

Answer

Use dot notation: obj.key or bracket notation: obj["key"].

22. How to add a new item to an array?

Answer

Use push() to add to the end, unshift() to add to the beginning.

23. What is destructuring in JS?

Answer

Destructuring is extracting values from arrays or objects into variables. Example: const [a, b] = [1, 2]
